# CirrusRubyCsv
Take Home Challenge

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'cirrus_ruby_csv'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install cirrus_ruby_csv

### To Run Application
    - Open `irb` in terminal
    - In irb
        - `require './lib/cirrus_ruby_csv.rb`
        - `DoIt.create_csv_and_report`

### Testing
    - run `rspec` in terminal
